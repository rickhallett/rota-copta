The Challenge

Thank you for taking the RotaCloud Angular challenge!

The task is to load data from two JSON files (using HTTP), relate the data, and display as per the example tables on both the users and roles pages. The final solution should look identical to the examples, there are no HTML or CSS changes needed.

The name column on both pages should consist of editable inputs. On the users page, the roles should be coloured as per their colour property and ordered by name. On the roles page, role names should be coloured accordingly and users should be ordered by name.

All changes to inputs should be reflected across both pages immediately, and both tables should be ordered by name and continue to order following updates.

Please complete the challenge in a GitHub repo and invite 'david-brandon' as a collaborator upon completion. If you have any questions, please email dev@rotacloud.com.

Challenge debrief notes

  1. Being completely honest, I would say this took me about 6 hours of effort, not 1.5. A significant chunk of that was spent going down the rabbit hole of asynchronous data services; it wasn't until I found out about BehaviourSubject and the data store pattern that I was more easily able to solve this problem (without modifying the HTML, which would have been a lot easier)
  2. If I had more time, I would like to add some unit tests, but as I have never written specs for Angular, it would hold my submission time back. Another potential improvement would be to statically map some of the data before the templates access it, as accessing this data via functions seems to generate a lot of component update cycles. For this example it doesn't matter, but I would imagine in a large app this could create a performance hit.
  3. Thank you for the opportunity!
